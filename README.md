# Build your unique map.

![01](/readme/01.jpg)

## This is a tool to create tiled map.

	1. Unlimited ranged.

	2. Export json format.

	3. Add any texture easyly.

## Expandability and instinctive UI.

	1. Floor texture.

	2. Walkable setting.

	3. Building objects.

	4. Light effect.

	5. Environment sound.

## How to start ?

	1. Open the file 'build/build_map.html'.

	2. Let create your map and fun.
